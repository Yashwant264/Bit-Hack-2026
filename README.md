# Bit Hack 2026 – Hackathon Website

A modern, full-stack hackathon website built for **BIT Hack 2026**, featuring a dynamic registration system, Firebase backend integration, and a secure admin dashboard.

---

## Live Project
👉 (https://bit-hack-2026.vercel.app/)

---

## Features

### Frontend
- Modern responsive UI  
- Smooth animations (GSAP)  
- 3D scroll effects  
- Mobile-friendly design  
- Clean and professional layout  

### Registration System
- Team size: 2–5 members  
- Collects:
  - Team Name  
  - College Name  
  - College Email ID  
  - Contact Number  
  - Member Details (Name, Registration No, Email)  
- Dynamic form generation  

### Backend
- Firebase Firestore integration  
- Real-time data storage  
- Structured data collection (`registrations`)  

### Admin Panel
- Secure login (Email + Password)  
- Firebase Authentication  
- View all team registrations  
- Search functionality  
- Displays full team data  

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Animations:** GSAP  
- **Backend:** Firebase Firestore  
- **Auth:** Firebase Authentication  
- **Deployment:** GitHub Pages / Vercel
